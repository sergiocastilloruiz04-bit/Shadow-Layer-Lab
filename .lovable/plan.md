

# GhostNet Landing Page

## Overview
A dark, high-tech landing page for GhostNet — a privacy-focused product suite designed for war correspondents and high-risk professionals. The tone is professional, secretive, and urgent.

## Design Direction
- **Dark theme** with deep blacks, dark grays, and accent colors in neon green or cyan for a cybersecurity/hacker aesthetic
- Subtle glitch effects, grid patterns, or scan-line overlays for atmosphere
- Monospace or sharp sans-serif typography
- Smooth scroll between sections

## Sections

### 1. Hero Section
- Bold headline: **"Invisible. Untraceable. GhostNet."**
- Subheadline emphasizing protection for war correspondents, whistleblowers, and high-risk professionals
- CTA button (e.g., "Enter the Network" or "Get Started")
- Subtle animated background (gradient pulse or particle-like effect)

### 2. Product Ecosystem
- **GhostNet Messaging** — featured card with "AVAILABLE NOW" badge, glowing/highlighted design, brief description of the secure messaging platform
- **GhostMail, GhostVPN, GhostDrive** — three cards with "COMING SOON" / "WORK IN PROGRESS" labels, faded/locked aesthetic with reduced opacity or lock icon overlay
- Each card has an icon, product name, short tagline

### 3. Pricing Section (GhostNet Messaging)
Three-column pricing layout:

- **Ghost Free ($0)** — Basic E2E encryption, secure messaging, standard metadata protection
- **Pro Journalist / Private** — Metadata obfuscation, Stealth UI (hidden app icon), Kill Switch, 0-Knowledge identity
- **Agency / Enterprise** — Fleet management, Secure Corporate Archives, 24/7 Priority Support, Asset Protection protocols

Each plan with a clear feature list and CTA button. The Pro plan visually highlighted as recommended.

### 4. Footer
- Links: Whitepaper, Manifest, "Join the Underground"
- Minimal, dark aesthetic with subtle separator
- Copyright line

## Technical Approach
- Single-page React app with smooth scroll navigation
- All content is static (no backend needed)
- Tailwind CSS with custom dark color scheme
- Lucide icons for product cards and feature lists
- CSS animations for atmospheric effects
